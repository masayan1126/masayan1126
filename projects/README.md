# projects/

注力プロジェクトの実態とワークフローを管理するディレクトリ。

「何に取り組んでいて、どう進めているか」を一箇所に整理する。

## プロジェクト一覧

|------|------------|--------|------|
| youtube | [YouTube チャンネル運営](youtube/overview.md) | 最優先 | AI・MCP・Claude Code 実践系の動画制作、収益化目標 |
| tech-blog | [テックブログ運営](tech-blog/overview.md) | 高 | 技術記事の執筆・公開、AdSense 収益化目標 |

| auto-news-collector | [完全自動情報収集](auto-news-collector/overview.md) | 高 | AI・生産性向上ニュースの完全自動収集・要約・配信 |

## 構成ルール

- 各プロジェクトには `overview.md`（概要・目標）と `workflow.md`（自動化フロー手順）を配置
- Draw.io で図を作成した場合は `<project>/docs/` に `.drawio.xml` で保存
- 新規プロジェクト追加時は `_template/` をコピーして作成
