# Chrome 拡張機能 ワークフロー

## フロー概要

アイデア → 開発 → Chrome Web Store 公開 → フィードバック反映のサイクル。

## ステップ

### 1. アイデア・設計

- **ツール**: idea-box、Claude Code
- **自動化**: 手動

### 2. 開発・テスト

- **ツール**: Claude Code CLI、ブラウザ
- **自動化**: 手動

### 3. 公開

- **ツール**: Chrome Web Store Developer Dashboard
- **自動化**: 手動

## 自動化構成

（現状はすべて手動フロー）
