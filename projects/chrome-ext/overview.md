# Chrome 拡張機能開発

## 概要

AI・生産性向上に関連する Chrome 拡張機能の開発・公開プロジェクト。

## 公開中の拡張機能

| 拡張名 | 概要 |
|--------|------|
| Article Date Detector | Web ページの公開日・更新日をポップアップ表示 |
| Markdown Preview | Markdown ファイルのリアルタイムプレビュー |
| Copy Tab Anything | タブのタイトルと URL を一括コピー |
| Web Annotator | Web ページにマーカーと付箋を追加 |
| Canvas Whiteboard | ブラウザ上のホワイトボード |
| Chat Stamp Preview | LINE スタンプ制作時のプレビュー |
