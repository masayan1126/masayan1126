# テックブログ ワークフロー

## フロー概要

技術的な学びや実践をブログ記事として整理し公開するフロー。YouTube 動画と連動したコンテンツ展開も行う。

## ステップ

### 1. テーマ選定

- **ツール**: idea-box / YouTube 動画テーマとの連動
- **自動化**: 手動

### 2. 記事執筆

- **ツール**: Claude Code、エディタ
- **自動化**: 手動

### 3. 公開・分析

- **ツール**: ブログ CMS、Google Analytics
- **自動化**: blog-analytics Action（計画中）

## 自動化構成

| 自動化 | 種類 | 状態 |
|--------|------|------|
| blog-analytics | GitHub Actions (daily) | 計画中 |
